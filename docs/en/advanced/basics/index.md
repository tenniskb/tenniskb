The Mastery Manual is dense and long. It's not meant to be read once and done. It's meant to be *worked* — read a section, take it to the court, come back next week and re-read the same section with new eyes.

The first read will feel like too much. The fifth read will feel obvious. That's the pattern with all deep material.
