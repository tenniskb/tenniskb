**A bilingual research library on tennis biomechanics, control theory, and stroke mechanics.**

This is the English version. The Vietnamese edition is available at [`/vi/`](../../vi/).

---

## About

This knowledge base synthesizes 30 research manuscripts covering tennis biomechanics, neurological control theory, kinetic chain mechanics, and stroke-specific technique. The work draws on elite-player analysis (Federer, Wawrinka, Dimitrov, Alcaraz, Sinner, Djokovic) and applies industrial control systems concepts (feedforward/feedback, IPS, NTRP coaching) to tennis technique.

The framework integrates:

- **Biomechanics**: kinetic chain, momentum transfer, contact-point architecture
- **Neurological control theory**: feedforward/feedback systems, CNS control layers
- **Eastern internal-martial-arts concepts**: Dantian, Mingmen, COG projection (Dương/Yin-Yang wave generation)
- **Industrial control systems**: PID-style layered control, Integrated Pro System (IPS)
- **Stroke-specific mechanics**: forehand, one-handed backhand, serve, volley, split step, stance comparison

## How to use this site

Each section links to a research document. Topics are grouped by stroke and by analytical framework. The site navigation is on the left.

If you are a coach or player:

1. Start with the **Synthesis** section for a 30-document overview
2. Then drill into a stroke category that interests you
3. Cross-reference EN and VI versions to see the same topic from both linguistic perspectives
4. The **NTRP 3.5 Coaching** section provides a coaching-system framework targeted at the 3.5 → 4.0 transition
